# Better Google Form Filler Privacy Policy

Last updated: 2026-04-22

This privacy policy explains how Better Google Form Filler handles data when you use the extension.

## What the extension does

Better Google Form Filler helps a user:

- connect to a Google account,
- read records from a user-selected Google Sheets spreadsheet,
- match spreadsheet columns to questions in an open Google Forms page, and
- fill the current form with a selected record.

## Data the extension handles

Depending on how you use the extension, it may handle the following data:

- Extension settings stored in `chrome.storage.sync`, such as:
  - spreadsheet ID,
  - sheet name,
  - search field preferences,
  - mapping override settings.
- Google account profile data returned by Google during sign-in, such as email address and display name.
- Data contained in the Google Sheets spreadsheet selected by the user.
- Data contained in the currently open Google Forms page, including question titles and field structure needed to fill the form.

## How the data is used

The extension uses this data only to provide its core functionality:

- authenticate the user with Google,
- read the selected spreadsheet,
- help the user search for a record,
- match spreadsheet fields to form questions, and
- fill the current Google Form on the user's device.

## How data is shared

The extension does not send user data to the developer's own servers.

Data is transmitted only as needed to Google services that are required for the extension to work, including:

- `docs.google.com`
- `sheets.googleapis.com`
- `www.googleapis.com`

The extension does not sell user data.

## Data retention

- Extension settings remain in Chrome sync storage until the user changes them, clears browser sync data, or removes the extension.
- Google authentication tokens are managed by Chrome and Google.
- Spreadsheet data and form page data are processed in the browser during use and are not stored by the developer.

## User control

Users can:

- change or delete saved extension settings,
- sign out or revoke Google access through their Google account settings,
- remove the extension at any time.

## Security

The extension communicates with Google services over HTTPS.

## Google API data

The use of information received from Google APIs will adhere to the Chrome Web Store User Data Policy, including the Limited Use requirements.
