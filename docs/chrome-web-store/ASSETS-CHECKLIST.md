# Chrome Web Store Asset Checklist

## Required / expected assets

Based on the current Chrome Web Store listing guidance, prepare:

- Store icon: `128 x 128`
- Screenshots: at least `1`, up to `5`, each `1280 x 800`
- Small promo tile: `440 x 280`
- Marquee promo tile: `1400 x 560` optional
- Promo video: optional YouTube URL

## Suggested screenshot plan

### Screenshot 1

Show the popup with:

- current page status,
- configured sheet source,
- actions to open the panel and refresh records.

### Screenshot 2

Show the options page with:

- spreadsheet URL or ID field,
- sheet selector,
- search field configuration,
- mapping override area.

### Screenshot 3

Show the in-page Google Forms panel with:

- searchable record list,
- visible field preview,
- clear "Apply" action.

### Screenshot 4

Show a real form being filled with:

- text inputs,
- radio buttons,
- checkboxes,
- dropdown selection.

### Screenshot 5

Show an advanced use case:

- mapping overrides,
- non-identical question labels,
- successful matching behavior.
