# Chrome Web Store Submission Pack

This folder contains the draft materials needed to publish Better Google Form Filler on the Chrome Web Store.

## Files

- [STORE-LISTING.md](./STORE-LISTING.md): draft listing copy for the Store listing tab
- [PRIVACY-TAB.md](./PRIVACY-TAB.md): suggested answers for the Privacy tab
- [ASSETS-CHECKLIST.md](./ASSETS-CHECKLIST.md): required image sizes and suggested screenshot plan
- [REVIEWER-NOTES.md](./REVIEWER-NOTES.md): draft reviewer instructions and test setup notes
- [SUBMISSION-CHECKLIST.md](./SUBMISSION-CHECKLIST.md): final pre-submit checklist

## Before you submit

Replace all placeholders:

- `<your-account>`
- `<your-repo>`
- `<your-support-email>`
- test spreadsheet and form URLs
- any final store copy that should reflect your actual branding
