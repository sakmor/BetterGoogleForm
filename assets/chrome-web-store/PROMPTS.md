# Generation Prompts

These are the core prompts used to generate the current Chrome Web Store asset pack.

## Store icon

Use case: `logo-brand`

Goal:

Create a square icon that combines:

- a spreadsheet grid,
- a form document,
- and a clear autofill direction.

Constraints:

- no letters,
- no logos,
- high legibility at small size,
- warm ivory + teal + amber palette.

## Small promo tile

Use case: `product-mockup`

Goal:

Create a `440x280` promo-style banner showing:

- spreadsheet data on the left,
- a search / picker panel in the middle,
- a filled form on the right,
- the headline `Search Sheets. Fill Forms.`

Constraints:

- strong readability,
- no browser chrome,
- no logos,
- clean Chrome Web Store marketing feel.

## Marquee promo tile

Use case: `product-mockup`

Goal:

Create a `1400x560` wide hero banner showing:

- spreadsheet records flowing into form-ready answers,
- a premium SaaS-style composition,
- the headline `From spreadsheet records to form-ready answers`

Constraints:

- clear left headline zone,
- strong workflow story,
- no people,
- no logos,
- no photorealism.
